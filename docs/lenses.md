# Линзы (Optics)

Линза — это доменная модель знания, описанная через операторы ProcessLang. Когда Ева использует линзу, она смотрит на любую задачу через призму конкретной области — и её мышление меняется вместе с этим.

---

## Как работают линзы

Линза — это Lua файл в папке `optics/` с определённой структурой:

```lua
local lens = {}
lens.name    = "Psychology"
lens.domain  = "Описание домена..."
lens.operators = {
    FLOW     = "как FLOW проявляется в этом домене",
    CONNECT  = "как CONNECT проявляется в этом домене",
    DISSOLVE = "...",
    ENCODE   = "...",
    CHOOSE   = "...",
    OBSERVE  = "...",
    CYCLE    = "...",
    LOGIC    = "...",
    RUNTIME  = "...",
    MANIFEST = "..."
}
return lens
```

При старте planGOD CONNECT загружает все линзы из `optics/` и инжектирует их в системный промпт. Ева видит каждую линзу как набор интерпретаций операторов ProcessLang в конкретном домене.

## Встроенные линзы

### addiction
Зависимость как процессуальная система подкрепления. Нейробиологические механизмы, циклы reward/withdrawal, паттерны деагентности.

### adhd
ADHD как нейродивергентный процессуальный профиль. Attention dysregulation, hyperfocus, executive function.

### agency
Агентность как способность инициировать процессы. Автономия, воля, ответственность, внутренний локус контроля.

### biology
Живые системы как процессуальные сети. Метаболизм, гомеостаз, эволюция, клеточные механизмы.

### chemistry
Химические реакции как процессы трансформации. Реакционная кинетика, термодинамика, катализ.

### code
Программирование как формализация процессов. Алгоритмы, структуры данных, архитектурные паттерны.

### crypto
Криптографические протоколы как процессы верификации доверия. Блокчейн, консенсус, шифрование.

### dissipativemath
Диссипативные структуры и нелинейная динамика. Аттракторы, бифуркации, самоорганизация.

### economics
Экономика как процессуальная система обмена. Рынки, стимулы, равновесие, институты.

### electricity
Электрические цепи как процессуальные потоки. Напряжение, ток, сопротивление, мощность.

### examplelens
Шаблон для создания новых линз.

### harmonicflow
Гармония, ритм и музыкальные процессы. Волны, резонанс, тональность.

### institutionaleducation
Образование как социальный институт. Системные цели, стандартизация, деагентность, скрытый учебный план.

### kabbalahiching
Синтез Каббалы и И-Цзин как процессуальная система. Древо Сефирот, 64 гексаграммы, эманация, синхронистичность.

### math
Математические структуры как процессы. Доказательство, абстракция, формализация.

### pedagogy
Обучение как процесс трансформации понимания. Зона ближайшего развития, scaffolding, рефлексия.

### psychology
Психика как процессуальная система. Когниция, эмоции, защитные механизмы, развитие.

### psychopathology
Психические расстройства как нарушения процессуальных паттернов. Дисфункциональные циклы, симптоматика.

### quantum
Квантовая механика как процессуальное описание реальности. Суперпозиция, запутанность, коллапс волновой функции.

### sociology
Социальные системы как процессуальные сети. Структуры, нормы, власть, воспроизводство.

## Создание линзы

Ева может создавать линзы самостоятельно. Достаточно попросить:

```
создай линзу для домена "термодинамика"
```

Ева сгенерирует Lua файл и сохранит его в `optics/`. При следующем запуске линза автоматически загрузится.

Можно также создать линзу вручную — скопируй `optics/examplelens.lua` и заполни операторы.

### Советы по написанию линз

**Каждый оператор должен отвечать на вопрос:** "Как этот оператор ProcessLang проявляется в данном домене?"

Например для домена "музыка":
- `FLOW` = поток звука, мелодическая линия, движение темпа
- `CYCLE` = ритм, такт, форма (ABA, сонатная форма)
- `DISSOLVE` = диссонанс, разрешение, паузы
- `MANIFEST` = исполнение, запись, концерт

**Описания должны быть конкретными**, не абстрактными. Не "процесс изменения" а "модуляция тональности через пивотный аккорд".

**Линза влияет на мышление Евы** — чем точнее описания, тем глубже анализ.

## Комбинирование линз

Ева может использовать несколько линз одновременно. Это один из главных инструментов planGOD — синтез через разные домены:

```
проанализируй зависимость от социальных сетей через линзы addiction и sociology
```

Ева применит операторы ProcessLang через призму обоих доменов и найдёт структурные параллели которые невидимы при взгляде из одной линзы.
